# claude-cowork-plugins — RETIRED

This repository is **retired and archived**. It is no longer the distribution route for the
Cornell Systems Engineering Claude plugins.

**Everything now lives in the private `systemsbot` repository.** All plugins ship as a single
download — `systemsbot-plugins.zip` — built and published automatically there.

- If you're a Cornell SysEng staff/faculty member: contact **Tim Fraser (tmf77@cornell.edu)**
  for the current plugin bundle and install instructions.
- Do **not** install anything from this repo's history — those builds are outdated and were
  missing the connector configuration Cowork/Claude Code needs, so their tools won't load.

_Archived 2026-07-13._
